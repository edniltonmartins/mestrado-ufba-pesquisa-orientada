Busca por software livre

Busca por software livre é uma das principais funções deste site. E para facilitar ainda mais a sua busca, iremos dar algumas dicas que podem ser de grande utilidade.

Buscar por nome

Para buscar um software específico, você pode utilizar a expressão abaixo no campo de busca: 

name:"nome do software"

Você também pode buscar pelo nome do projeto utilizando uma palavra chave qualquer ao invãs de uma expressão:

name:palavrachave

No resultado da busca irão aparecer, dentre os projetos cadastrados, apenas aqueles cujo nome contiver a expressão entre aspas ou palavra chave informada. Veja abaixo um exemplo:

Buscar por descrição

Do mesmo modo que foi feito para o nome do projeto, você também pode buscar apenas pela descrição do projeto utilizando comandos semelhantes: 

description:"expressão qualquer a ser buscada na descrição do projeto"
description:palavrachave

Buscar por palavras chave

De modo semelhante, ainda é possível efetuar buscas apenas nas palavras chave associadas ao projeto de software livre, utilizando o comando abaixo: 

tags.name:palavrachave

Como resultado, serão exibidos apenas projetos de software livre que contenham a palavra chave associada. É possível que a palavra chave não apareça no resultado da busca (alguns projetos possuem mais de 7 palavras chave associadas). Neste caso, para visualizar todas as palavras chave de um projeto é só acessar a página de detalhe, clicando no nome do projeto.

Projetos Recomendados

Quando um usuário visualiza a página de detalhe de um projeto, nós armazenamos a identificação do usuário, o projeto e o horário em que o projeto foi visualizado. Nós utilizamos estes dados com o único propósito de recomendar projetos de software livre para os outros usuários do site.

Visualizando projetos recomendados

Para visualizar projetos recomendados, você deve entrar no site com o seu usuário e senha. Caso você ainda não possua usuário cadastrado, é necessário criá-lo. Você pode fazer isso acessando os botões na parte superior direita da tela.

Filtrando projetos recomendados

Caso você deseje, é possível filtrar as recomendações utilizando as próprias palavras chave associadas aos projetos. Para isso, após entrar no site com o seu usuário e senha, vá até o menu no canto superior direito da tela e clique no seu nome de usuário, e logo após clique em "Configurações".

Na tela de configurações para o seu usuário, clique em "Tags de Interesse".

Na tela que aparece, pode-se visualizar as tags (palavras chave) que você já utiliza como filtro. Há um campo texto no qual você pode digitar palavras chave. Ao digitá-las, o site efetuará uma busca entre as palavras chave cadastradas, para que você selecione aquelas que mais se aproximam do seu interesse. Após ter selecionado todas as palavras chave desejadas, clique em adicionar.

Voltando à página inicial, você pode visualizar os projetos que nós lhe recomendamos, utilizando as palavras chave selecionadas como filtro

Avaliação de projetos

Você pode (e recomendamos fortemente que o faça) avaliar projetos de software livre que já tenha utilizado em algum momento. As avaliações dos projetos de software livre feitas por você auxiliam os outros usuários a terem uma noção de quão útil pra eles estes projetos podem ser.

Avaliando um projeto

Para avaliar um projeto de software livre em nosso site, primeiro acesse a página de detalhe dele. Lá é possível visualizar alguns dados básicos do projeto que podem ser úteis para uma avaliação mais coerente.
