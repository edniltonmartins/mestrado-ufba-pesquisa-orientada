Busca por software livre

Busca por software livre é uma das principais funções deste site. E para facilitar ainda mais a sua busca, iremos dar algumas dicas que podem ser de grande utilidade.

Buscar por nome

Para buscar um software específico, você pode utilizar a expressão abaixo no campo de busca: 

name:"nome do software"

Você também pode buscar pelo nome do projeto utilizando uma palavra chave qualquer ao invãs de uma expressão:

name:palavrachave

No resultado da busca irão aparecer, dentre os projetos cadastrados, apenas aqueles cujo nome contiver a expressão entre aspas ou palavra chave informada. Veja abaixo um exemplo:

Buscar por palavras chave

De modo semelhante, ainda é possível efetuar buscas apenas nas palavras chave associadas ao projeto de software livre, utilizando o comando abaixo: 

tags.name:palavrachave

Como resultado, serão exibidos apenas projetos de software livre que contenham a palavra chave associada. É possível que a palavra chave não apareça no resultado da busca (alguns projetos possuem mais de 7 palavras chave associadas). Neste caso, para visualizar todas as palavras chave de um projeto é só acessar a página de detalhe, clicando no nome do projeto.

Buscar por licença de uso

Ainda há a possibilidade de se efetuar buscas utilizando licenças de uso dos projetos de software livre como critério de filtro, utilizando o comando abaixo: 

tags.name:licençaDeUso

Como resultado, serão exibidos apenas os projetos de software livre que possuam a licença de uso especificada. Para verificar a licença de uso, visualize a página de detalhe do respectivo projeto.

Projetos Recomendados

Quando um usuário visualiza a página de detalhe de um projeto, nós armazenamos a identificação do usuário, o projeto e o horário em que o projeto foi visualizado. Nós utilizamos estes dados com o único propósito de recomendar projetos de software livre para os outros usuários do site.

Visualizando projetos recomendados

Para visualizar projetos recomendados, você deve entrar no site com o seu usuário e senha. Caso você ainda não possua usuário cadastrado, é necessário criá-lo. Você pode fazer isso acessando os botões na parte superior direita da tela.

Filtrando projetos recomendados

Caso você deseje, é possível filtrar as recomendações utilizando as próprias palavras chave associadas aos projetos. Para isso, após entrar no site com o seu usuário e senha, vá até o menu no canto superior direito da tela e clique no seu nome de usuário, e logo após clique em "Configurações".

Na tela de configurações para o seu usuário, clique em "Tags de Interesse".

Na tela que aparece, pode-se visualizar as tags (palavras chave) que você já utiliza como filtro. Há um campo texto no qual você pode digitar palavras chave. Ao digitá-las, o site efetuará uma busca entre as palavras chave cadastradas, para que você selecione aquelas que mais se aproximam do seu interesse. Após ter selecionado todas as palavras chave desejadas, clique em adicionar.

Voltando à página inicial, você pode visualizar os projetos que nós lhe recomendamos, utilizando as palavras chave selecionadas como filtro

Avaliação de projetos

Você pode (e recomendamos fortemente que o faça) avaliar projetos de software livre que já tenha utilizado em algum momento. As avaliações dos projetos de software livre feitas por você auxiliam os outros usuários a terem uma noção de quão útil pra eles estes projetos podem ser.

Avaliando um projeto

Para avaliar um projeto de software livre em nosso site, primeiro acesse a página de detalhe dele. Lá é possível visualizar alguns dados básicos do projeto que podem ser úteis para uma avaliação mais coerente. A opção para avaliar um projeto está localizada logo abaixo da avaliação da comunidade para o projeto.

Ao clicar nesta opção, uma nova tela disponibilizará uma série de critérios para que você avalie o software. São eles:

  1. Facilidade de entender, testar ou customizar
  2. Desempenho/Eficiência
  3. Atende às necessidades
  4. Facilidade de instalação/configuração
  5. Qualidade da documentação
  6. Custo Total de Propriedade de Software
  7. Relevância/Utilidade
  8. Existência de Suporte Externo
  9. Curva de aprendizado para a utilização da solução
  10. Popularidade

Caso você não tenha entendido algum destes critérios, você pode clicar no ícone , e uma breve descrição do critério de avaliação lhe será exibida. Clique novamente no ícone para que a dica seja fechada.

Você pode escolher entre 1 e 5 estrelas para cada critério. O significado de cada quantidade de estrelas irá aparecer logo abaixo do critério de avaliação, como exemplificado abaixo:

Você pode deixar critérios em branco, caso não saiba como avaliá-lo. Após dar uma nota (ou deixar em branco) para os critérios de avaliação, dê uma avaliação geral para o software.

Caso você tenha algo mais a dizer sobre o software livre, deixe uma opinião textual. Ela será bastante útil para os demais potenciais usuários. Evite utilizar palavras de baixo calão, xingamentos e afins. Ao término da sua avaliação para o software, clique no botão enviar.

Visualizando avaliações de um projeto

Ao entrar na página de detalhes do projeto, você poderá visualizar a média das avaliações dadas pela comunidade de usuários do site. Também é possível visualizar a média das avaliações para cada critério. Caso tenha dúvidas sobre o significado de cada critério, você pode clicar no ícone de ajuda , do mesmo modo como na avaliação do projeto.

Você também pode visualizar as opiniões dos usuários sobre cada projeto, clicando no link Veja outras avaliações.

A página que é apresentada logo a seguir é a mesma que lhe é apresentada após o término da avaliação de um projeto. Na parte superior são exibidos dados básicos sobre a média das avaliações e a quantidade de avaliações. No meio da tela aparecem a média das avaliações por critério, como na página de detalhe. Logo abaixo, aparece a lista de opiniões deixadas pelos usuários.

Clicando nos links Opiniões mais úteis ou Opiniões mais recentes, uma lista das opiniões dos usuários em forma de timeline é exibida.

Visualizando as suas próprias avaliações

Para visualizar todas as avaliações que você tenha feito para os projetos cadastrados no site, clique no seu login, na parte superior direita da tela. Escolha a opção Perfil. Na tela que aparece, clique no link Opiniões, na parte inferior direita da tela.

Uma listagem das suas opiniões para os projetos será exibida, também em forma de timeline.
