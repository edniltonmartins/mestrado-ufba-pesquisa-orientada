Sobre

Afinal de contas, o que é esse site?

Bom, pra começar, este é um trabalho de mestrado que visa auxiliar a comunidade a conhecer melhor software livre. Cada opinião sobre um determinado software é baseada em uma série de critérios de avaliação, que refletem o que a maioria dos usuários considera importante para avaliar a qualidade de um software.

Tudo bem, mas como vocês pretendem fazer isso?

Com base nas opiniões que você e outros usuários dão sobre os softwares livres que estão aqui indexados, e dos critérios de avaliação que nós selecionamos, queremos entender qual a melhor maneira de recomendar software. Poderíamos fazer recomendações com base em usuários que avaliam os mesmos softwares que você, com base nos softwares que foram avaliados de modo semelhante aos que você avaliou, ou ainda avaliando as propriedades de cada software. Entretanto, ainda estamos estudando as possibilidades.

Beleza! E depois?

A nossa intenção é coletar esses dados, definir uma técnica para recomendar software livre, e escrever publicações sobre este trabalho, além da dissertação de mestrado, é claro. Então, ajude-nos! :) Se puder, convide seus amigos a nos ajudar, opinando sobre cada software livre que eles conhecem, quanto mais gente, melhor.
